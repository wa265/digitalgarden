## 开源阅读简介与使用教程


**Legado（阅读）** 是一款 **Android 平台的免费开源小说阅读工具**，主打 **自定义书源、订阅网络内容、本地 TXT/EPUB 阅读** 和 **高度可定制的阅读界面**。它本身**不直接提供内容**，而是由用户自行导入书源或订阅源来获取网络文学内容；同时支持广告净化、字体与排版调整、多种翻页模式等功能。很遗憾在2024年由于腾讯与网络的严打，开发者坤飞已停止维护项目，但软件的主体功能基本完善，书源也有大佬不断进行维护，作为日常的阅读使用也非常稳定。

1. 软件下载
访问[开源阅读 github 网址](https://github.com/aoaostar/legado)，下载apk安装包，例如 `legado_app_3.23.040420.apk`，然后点击安装即可。如果网络不支持访问 Github 也可以下载我提供的网盘链接[开源阅读网盘版本](https://share.fnnas.net/s/0b6cdb642c7f497497)

2. 书源导入
浏览器访问[喵公子订阅源](http://dy.miaogongzi.cc/) ，网页里点击喵公子订阅源，直接跳转到app打开，完成。  
如果第一个订阅源打不开，可以导入第二个备用订阅源。
- 打开软件，点击最后一个图标**我的**
- 点击**书源管理**
- 点击右上角三个小点
-  选择**网络导入**
- 复制粘贴书源，确定
- **确定**导入书源
- 选择第一个图标**书架**
- 点击书架右上角**搜索**图标，搜索想要阅读的书籍即可

[喵公子网站](https://fb.miaogongzi.cc/)
[源仓库](https://yckceo.vip)
3. 格局排版推荐
不同的阅读排版可以从酷安的阅读的话题中进行下载和导入，根据自己的喜好进行调整
